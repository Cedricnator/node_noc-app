
# NOC

## Description:
Monitoring application(logs) with clean architecture.

## Install:
1. Clone the repository
```
git clone https://github.com/Cedricnator/node_noc-app.git
```
2. Install dependencies
```
npm install
```
3. Run
```
npm run dev
```
4. Run Test
```
npm run test
```
